# .storetle browser viewer (WebAssembly demo)

A zero-framework single-page viewer for `.storetle` archives. The Rust reader
(`../rust/`) is compiled to wasm with the **pure-Rust `ruzstd` backend** — no
zstd-sys/C code in the wasm build.

**Dictionary support on wasm: fully working.** `ruzstd` ≥ 0.8 decodes
ZDICT-format dictionaries (`Dictionary::decode_dict` + frame dictionary-id
lookup). This was verified by running the full Python-differential test suite
against the ruzstd backend (`cargo test --no-default-features --features
zstd-pure`) and by a Node smoke test of the actual wasm artifact: both
external-dictionary archives (with `cube_dict_v10.bin`) and embedded-dictionary
archives decode byte-identically to the Python reference. No zstd-sys-on-wasm
fallback was needed.

## Run

```sh
cd web
python3 -m http.server 8000
# open http://localhost:8000/
```

(Any static server works; a server is required because ES modules + wasm
don't load from file:// URLs.)

What you can do:

- **Load sample** — loads `sample.storetle` (5 docs, embedded dictionary;
  generated by `../rust/tests/gen_fixtures.py`).
- **Drop / choose a `.storetle` file.** If the file does not embed its
  dictionary, the page needs `cube_dict_v10.bin`: it auto-fetches
  `./cube_dict_v10.bin` (a copy is included here), or you can drop/choose a
  dictionary file manually.
- Click a doc → rendered in a **sandboxed iframe** (`sandbox` attribute with
  no tokens: no scripts, no same-origin, no forms) via `srcdoc`.
- **View source** toggles the raw reconstructed HTML.

## Rebuild the wasm package

Requires `wasm-pack` (`brew install wasm-pack` or `cargo install wasm-pack`)
and the `wasm32-unknown-unknown` target (`rustup target add
wasm32-unknown-unknown`):

```sh
cd ../rust/storetle-wasm
wasm-pack build --target web --release --out-dir ../../web/pkg
```

This produces `pkg/storetle_wasm.js` (ES-module glue) and
`pkg/storetle_wasm_bg.wasm` (~130 KB), which `index.html` imports directly.

## Files

| file | what |
|---|---|
| `index.html` | the whole app (markup, styles, JS module) |
| `pkg/` | wasm-pack output (checked in so the demo runs without a Rust toolchain) |
| `sample.storetle` | demo archive with embedded dictionary |
| `cube_dict_v10.bin` | codec dictionary, served so the page can pre-fill it |
